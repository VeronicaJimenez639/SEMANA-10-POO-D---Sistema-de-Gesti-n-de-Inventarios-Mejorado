# Sistema de Gestión de Inventario

Sistema desarrollado en Python utilizando Programación Orientada a Objetos para gestionar los productos de un almacén.

## Funcionalidades

- Añadir productos
- Eliminar productos
- Actualizar cantidad o precio
- Buscar productos por ID
- Listar inventario

## Mejoras - Semana 10

- Persistencia en archivo inventario.txt
- Carga automática de datos al iniciar
- Manejo de excepciones
- Creación automática de carpeta y archivo si no existen
